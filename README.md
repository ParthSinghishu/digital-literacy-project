## 📌 Project Overview

This repository contains my Digital Literacy Portfolio developed as part of the CSE0001 course. As a Student Digital Ambassador, the aim of this project is to help students understand and navigate the digital world effectively.

The project focuses on building awareness about:
- Digital literacy fundamentals  
- Professional online presence  
- Coding and collaboration platforms  
- Email communication etiquette  
- Cybercrime awareness and prevention  

All tasks are organized into structured folders with relevant screenshots, documents, and reflections.
